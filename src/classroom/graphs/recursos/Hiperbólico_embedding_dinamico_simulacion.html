Embedding dinámico

Una simulación visual para entender cómo objetos discretos o de alta dimensión se transforman en vectores densos, donde la distancia geométrica representa similitud semántica, estructural o funcional.

1. Espacio de entrada

Cada objeto comienza como una entidad discreta. El embedding aplica una función φ que lo proyecta a un espacio vectorial continuo de baja dimensión.
