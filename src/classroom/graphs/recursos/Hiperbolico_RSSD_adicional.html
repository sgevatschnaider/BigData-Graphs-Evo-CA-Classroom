RSGD en el disco de Poincaré

Esta simulación muestra cómo un punto aprende su posición dentro de un espacio hiperbólico usando Riemannian Stochastic Gradient Descent. La diferencia central con SGD tradicional es que el paso de optimización no se toma como si el espacio fuera plano: primero se adapta el gradiente a la métrica riemanniana y luego se retrae el punto para permanecer dentro del disco.

SGD euclídeo: espacio plano
xt+1 = xt − η ∇f(xt)
RSGD: espacio curvo
xt+1 = Rxₜ(−η gradR f(xt))

Qué está pasando

El punto azul se mueve dentro del disco de Poincaré. En cada paso se elige una referencia objetivo, se calcula una dirección de ajuste y se corrige el movimiento para respetar la geometría hiperbólica.

En el borde del disco, la distancia hiperbólica crece mucho. Por eso un pequeño movimiento visual puede representar un gran cambio geométrico. RSGD evita tratar el disco como si fuera un plano euclídeo común.
