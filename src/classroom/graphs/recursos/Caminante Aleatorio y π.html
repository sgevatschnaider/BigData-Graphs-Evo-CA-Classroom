Key Concept

El código usa el retorno al origen de un caminante aleatorio sobre el grafo ℤ² para estimar π a partir de leyes asintóticas.

Key Concept Explanation

La idea central es que un paseo aleatorio simple en una red bidimensional no se comporta de manera arbitraria cuando el tiempo crece. Existe una ley asintótica precisa para la probabilidad de retorno al origen, y esa ley involucra a π. Por eso se puede simular el proceso muchas veces, medir retornos y reconstruir numéricamente el valor de π. Este es un ejemplo muy interesante de cómo la teoría de grafos no solo sirve para estudiar conexiones estáticas, sino también procesos dinámicos sobre redes. El caminante aleatorio transforma un grafo en un sistema probabilístico vivo, y el análisis asintótico permite extraer información matemática profunda de ese movimiento.
