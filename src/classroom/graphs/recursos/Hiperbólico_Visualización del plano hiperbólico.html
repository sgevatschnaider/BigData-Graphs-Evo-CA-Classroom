¿Se puede visualizar el plano hiperbólico?

Sí. El plano hiperbólico no se puede dibujar de manera perfecta en el plano euclidiano común, pero se puede representar mediante modelos. Cada modelo conserva algo distinto: ángulos, rectas, geodésicas o estructura algebraica.

2.20
Longitud hiperbólica 0 CH²(r) = 2π sinh(r)
Área hiperbólica 0 AH²(r) = 2π(cosh(r) − 1)
Longitud euclidiana 0 CR²(r) = 2πr
Área euclidiana 0 AR²(r) = πr²

1. Disco de Poincaré

Idea: todo el plano hiperbólico se dibuja dentro de un disco. El borde no pertenece al espacio: representa el infinito hiperbólico.
ρ = tanh(r / 2)
Si r crece, ρ se acerca al borde. Visualmente parece comprimido, pero en la métrica hiperbólica hay muchísimo espacio cerca del borde. Las geodésicas son diámetros o arcos que cortan el borde en ángulo recto.
Modelo conforme

2. Semiplano superior

Idea: el plano hiperbólico se representa como la región y > 0. La frontera y = 0 no pertenece al espacio.
H² = {(x, y) ∈ R² : y > 0}
Las geodésicas son rectas verticales o semicircunferencias perpendiculares a la frontera. Al igual que el disco de Poincaré, conserva ángulos.
Geodésicas rectas

3. Modelo de Klein

Idea: también usa un disco, pero ahora las geodésicas se ven como segmentos rectos o cuerdas.
ρKlein = tanh(r)
Es muy útil para ver caminos mínimos como líneas rectas. La desventaja es que no conserva bien los ángulos: las formas se deforman angularmente.
Modelo algebraico

4. Hiperboloide

Idea: el plano hiperbólico puede verse como una hoja de un hiperboloide en un espacio tridimensional con métrica lorentziana.
x² + y² − z² = −1,   z > 0
Este modelo es importante para la formulación matemática de embeddings hiperbólicos, porque permite trabajar con coordenadas y distancias de forma más algebraica.

¿Por qué esto ayuda a entender árboles y jerarquías?

En el plano euclidiano, el área disponible dentro de una bola de radio r crece como πr². En cambio, en el plano hiperbólico crece como 2π(cosh(r) − 1), que para radios grandes se comporta aproximadamente como una exponencial.

Esto es importante porque un árbol regular también crece exponencialmente por niveles: si cada nodo tiene b hijos, el nivel ℓ tiene aproximadamente b^ℓ nodos. Por eso el plano hiperbólico es una geometría muy natural para visualizar jerarquías, taxonomías, grafos arborescentes y redes con estructura centro-periferia.
